# Installation
1. Install [VScode](https://code.visualstudio.com/download)
1. Clone this repo
1. Install [Node.js](https://nodejs.org/en)
1. Install the following packages using CLI:
```
# three.js
npm install --save three

# vite
npm install --save-dev vite
```


# Run
```
cd <repo-path>
npx vite
```
